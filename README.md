# COMP3133 Assignment 1 - GraphQL Employee Management API

## Description
This project is a **GraphQL API** for managing users and employees.  
It allows you to **register and login users**, and **perform CRUD operations on employee data**, including searching by department or id.  
The backend is built with **Node.js, Express, Apollo Server, MongoDB, and Mongoose**, and uses **Cloudinary** for storing employee photos.

---

## Features

### Users
- Register a new user
- Login user

### Employees
- Add a new employee
- Update employee details
- Delete an employee
- Retrieve all employees
- Search employees by ID, department, or designation
